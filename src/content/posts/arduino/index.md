## ¿Qué es un Microcontrolador (MCU)?

El microcontrolador (MCU) es, en esencia, un pequeño ordenador contenido en un solo chip (circuito integrado). Está diseñado para gestionar tareas específicas dentro de un sistema embebido, sin necesidad de un sistema operativo complejo

Se le considera el "cerebro" del sistema. En su interior, un microcontrolador integra todos los elementos informáticos necesarios:

- **Unidad Central de Procesamiento (CPU)**: El núcleo encargado de ejecutar instrucciones y controlar las operaciones.

- **Memoria**: Incluye memoria volátil (RAM) para datos temporales y memoria no volátil (Flash/EEPROM) para almacenar el programa o firmware que el microcontrolador debe ejecutar.

- **Periféricos de Entrada/Salida (E/S)**: Componentes auxiliares como temporizadores, contadores, Convertidores Analógico-Digitales (ADC) y Digital-Analógicos (DAC), y protocolos de comunicación (UART, SPI, I2C).

Los MCU son ideales para el procesamiento de señales en tiempo real, el control de motores y la interacción con sensores. Operan a velocidades de reloj más bajas que los microprocesadores (hasta 200 MHz, en comparación con más de un GHz) y están optimizados para la eficiencia y el bajo consumo de energía.
